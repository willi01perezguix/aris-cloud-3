# PR Checklist v1.1.0

Usar este checklist en cada PR de la etapa v1.1.0.

- [ ] Objetivo del PR acotado (PR pequeño por objetivo).
- [ ] Evidencia before/after incluida.
- [ ] Pruebas ejecutadas (unit + smoke corto relevante).
- [ ] Riesgos identificados.
- [ ] Plan de rollback explícito.
- [ ] Impacto en permisos/scope documentado.
- [ ] Validación de no-regresión en flujo Tenant/Store/User.
- [ ] Si hay cambio de contrato API, versión explícita + documentación de contrato.
- [ ] Si aplica, feature toggle definido (activación/desactivación).
