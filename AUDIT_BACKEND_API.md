# Auditoría Forense Backend API (ARIS Cloud 3)

## Limitaciones de fuente
- No fue posible acceder al OpenAPI remoto (`https://aris-cloud-3-api-pecul.ondigitalocean.app/openapi.json`) por `403 Forbidden` de proxy en este entorno.
- Se usó como inventario principal el OpenAPI generado localmente desde `app.main:app.openapi()` y se comparó contra `artifacts/release_candidate/openapi.json`.

## A. RESUMEN EJECUTIVO
- **core_modules**: auth; admin (tenants/stores/users); stock; transfers; pos_sales; reports; exports; health/ready
- **suspicious_modules**: access_control hidden endpoints; ops metrics; assets upload image; pos_returns quote/actions
- **duplicates**: POST /aris3/auth/change-password vs PATCH /aris3/auth/change-password; /aris3/auth/login vs /aris3/auth/token (different contract, overlapping objective); reports vs exports (same data domain, different delivery mode)
- **external_validation_needed**: hidden access-control tenant/platform policy endpoints; ops metrics endpoint consumer; assets/upload-image external clients; stock migration/import endpoints used by external batch tools
