2. **Observabilidad distribuida:** `telemetry_hook` cubre proceso local; integración con collector externo queda para iteración posterior.
3. **Smoke sintético:** valida contrato del SDK y packaging, pero no reemplaza pruebas e2e de negocio completas.

## Rollback (si aplica)
